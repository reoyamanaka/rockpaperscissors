## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* wxPython
```sh
pip install wxpython
```


### Installation

1. Clone the repository
```sh
git clone https://github.com/reoyamanaka/rockpaperscissors.git
```
2. Install wxPython module
```sh
pip install wxpython
```
